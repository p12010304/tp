# Business to Business for You (B2B4U)

[![CI Status](https://github.com/AY2526S2-CS2103T-T08-1/tp/workflows/Java%20CI/badge.svg)](https://github.com/AY2526S2-CS2103T-T08-1/tp/actions)

![Ui]({{ baseUrl }}/images/Ui.png)

**B2B4U is a desktop application for managing your contact details.** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using AddressBook, head over to the [_Quick Start_ section of the **User Guide**]({{ baseUrl }}/user-guide/quick-start.html).
* If you are interested about developing AddressBook, the [**Developer Guide**]({{ baseUrl }}/developer-guide/index.html) is a good place to start.
